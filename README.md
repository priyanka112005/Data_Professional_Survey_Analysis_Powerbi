# Data_Professional_Survey_Analysis_Powerbi
#overview:
Analyzed survey responses from 630 data professionals worldwide to uncover trends in salaries, job satisfaction, programming language preferences, and career entry difficulty across roles like Data Scientist, Data Analyst, Data Engineer, and more.

#Data Source:
Primary data: Survey dataset (Excel) containing 630 responses with 13+ variables including job title, yearly salary (USD), country, age, programming language preference, happiness ratings (salary, work-life balance, coworkers, management), and career switch history.

#Key Insights:
Data Scientists earn the highest average salary, followed by Data Engineers and Data Architects — Data Analysts rank 5th
Python dominates as the favourite programming language across all roles
42.7% found breaking into data "Neither Easy nor Difficult", while 24.76% found it difficult
Work-life balance satisfaction (5.74/10) is notably higher than salary satisfaction (4.27/10) — professionals are underpaid relative to lifestyle
USA and India are the top two countries represented in the survey
Average age of survey takers is 29.87 years — a young, early-career-dominated field

#Tools Used:
Power BI · Excel
